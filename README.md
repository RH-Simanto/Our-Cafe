## 📖 About the Project

**Our Café** is a web-based café management system developed using **PHP, MySQL, HTML, CSS, and JavaScript**. The system allows customers to browse products, place orders, make table bookings, and contact the café online.

An **Android WebView application** has also been developed using Android Studio to provide mobile access to the website.

## ⚙ Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/our-cafe.git
```

2. Copy the project into your web server (`htdocs` for XAMPP).

3. Import `database.sql` into MySQL.

4. Update the database credentials in `config.php`.

5. Start Apache and MySQL.

6. Open your browser:

```
http://localhost/our-cafe
```
